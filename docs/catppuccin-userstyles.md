# Catppuccin Userstyles

Catppuccin is a community-driven theme system that supports many applications including userstyles for websites.

## Installation

### Prerequisites

Install the **Stylus** browser extension:
- [Chrome](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
- [Firefox](https://addons.mozilla.org/en-GB/firefox/addon/styl-us/)

> **Note:** If using Chrome, enable "Allow access to file URLs" in Stylus settings.

> **Tip:** If you use Dark Reader, disable it on websites with Catppuccin userstyles to avoid conflicts.

## Installing Userstyles

### Option 1: All Userstyles (Recommended)

1. Download [`import.json`](https://github.com/catppuccin/userstyles/releases/download/all-userstyles-export/import.json)
2. Open Stylus "Manage" page
3. Click **Import** in the Backup section
4. Select the downloaded `import.json` file

**Customization:** Use [@uncenter's Customizer](https://catppuccin-userstyles-customizer.uncenter.dev/) to customize accent color and flavor before downloading.

### Option 2: Individual Userstyles

1. Enable CSP Patching in Stylus **Settings > Advanced**
2. Click the **Stylus Install** badge on the desired userstyle's README

## Configuration

Each userstyle supports:
- **Light flavor** - light theme variant
- **Dark flavor** - dark theme variant  
- **Accent color** - main highlight color

## Self-hosted/Decentralized Apps

1. Open Stylus "Manage" page
2. Click on the userstyle name
3. Go to **Style settings**
4. Add instance URLs to **Custom included sites**, e.g., `https://example.org/*`
